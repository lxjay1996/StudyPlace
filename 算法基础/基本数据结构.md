## 1 链表与邻接表

### 1.1 单链表

- 数组模拟的链表称为静态链表

```cpp
// head存储链表头
//e[]存储节点的值
//ne[]存储节点的next指针
//idx表示当前用到了哪个节点
int head, e[N], ne[N], idx;

// 初始化
void init() {
    head = -1;
    idx = 0;
}

// 头插
void insertTohead(int a) {
    e[idx] = a, ne[idx] = head, head = idx ++ ;
}
// 同理，将x插到下标是k的点后面, 0~k
void insert(int k, int x) {
    e[idx] = x, ne[idx] = ne[k], ne[k] = idx++;
}

// 将头结点删除，需要保证头结点存在
void remove() {
    head = ne[head];
}
// 同理， 将下标是k的点后面的点删掉
void remove(int k) {
    ne[k] = ne[ne[k]];
}
```

### 1.2 双链表

- 可以不要头结点，让0号点作为头结点，1号点作为尾结点，这样便就有了两个哨兵

```cpp
// e[]表示节点的值
//l[]表示节点的左指针
//r[]表示节点的右指针
//idx表示当前用到了哪个节点
int e[N], l[N], r[N], idx;

// 初始化
void init() {
    //0是左端点，1是右端点
    r[0] = 1, l[1] = 0;
    idx = 2;
}

// 在下标是k的点的右边，插入x
void insert(int a, int x) {
    e[idx] = x;
    l[idx] = a, r[idx] = r[a];
    l[r[a]] = idx, r[a] = idx ++ ;
}

// 删除第k个点
void remove(int a) {
    l[r[a]] = l[a];
    r[l[a]] = r[a];
}
```

### 2.3 单调栈与单调队列

1. 单调栈

   - 常见模型：找出每个数左边离它最近的比它大/小的数
   - 先暴力法，然后看能不能发现一些性质，比如单调性

    ```cpp
    int tt = 0;
    for (int i = 1; i <= n; i ++ ) {
        // q[i]入栈前,要把加入后不满足单调性的元素出栈
        while (tt && check(stk[tt], q[i])) tt -- ;
        // 具体问题的逻辑
        stk[ ++ tt] = q[i];    // q[i] 入栈
    }
    ```

2. 单调队列

   - 常见模型：找出滑动窗口中的最大值/最小值
   - 先考虑普通队列的暴力做法，然后进行优化，将队列中的没有用的元素剔除，找到单调性，这样求极值就O(1)

    ```cpp
    // q[N]中存放的是a[N]的元素下标
    int hh = 0, tt = -1;
    for(int i=0; i<n; i++) {
        // 判断队头是否在窗口里面
        while(hh <= tt && checkOut(q[hh])) hh++;

        while(hh <= tt && check(q[tt], a[i])) tt--;
        q[++tt] = i;    // a[i]元素下标入队

    }
    ```

```cpp
// s[] 是长文本，p[]是模式串，n=s.size(), m=p.size()

for(int i=2, j=0; i<=m; i++) {

    if(j == m) {

    }
```

## 4 Trie

- 高效的**存储和查找字符串集合** 的数据结构
- 题型：要么都是小写字母，要么都是大写字母，要么都是数字，反正字母数量肯定不多

```cpp
// idx表示下标，就是节点号，0号节点既是根节点，又是空节点
// son[N][] 存储树中每个结点的子节点
// cnt[]存储以每个子节点结尾的单词数量
int son[N][26], cnt[N], idx;
// 插入一个字符串
void insert(char str[]) {
    int p = 0;  // 从根节点开始
    for(int i=0; str[i]; i++) { // 遍历字符串
        int u = str[i] - 'a';   // 计算字符的相对索引，映射为数字
        // 如果p节点的子节点u不存在，那么就创建p的子节点u
        if(!son[p][u]) son[p][u] = ++idx;   // 给创建的每个节点打上号
        p = son[p][u];  // 没有就创建，说明p的子节点一定存在，更新p为其子节点，直到走到字符串尽头
    }
    cnt[p]++;   // 在这个字符串遍历完后，记录以最后一个子节点结尾的单词数+1
}

// 查询字符串出现的次数
int query(char *str) {
    int p = 0;
    for(int i=0; str[i]; i++) { // 遍历字符串
        int u = str[i] - 'a';
        if(!son[p][u]) return 0;    // 如果某个节点不存在对应的子节点，说明该字符串不存在集合中
        p = son[p][u];
    }
    return cnt[p];  // 走到最后一个子节点，说明存在该字符串，直接返回数量
}
```

## 5 并查集 -- 重要：短而精

- 将两个集合合并
- 询问两个元素是否在一个集合中
- 原理： 每个集合用一个树来表示。 树根的编号就是整个集合的编号。每个节点存储它的父节点，p[x]表示x的父节点。**使用并查集可以维护一些额外的信息** ，因此而变得强大
  - 问题1；如何判断树根：if(p[x] == x)
  - 问题2：如何求x的集合编号：while(p[x] != x) x = p[x];
  - 问题3：如何合并两个集合：px是x的集合编号，py是y的集合编号，p[x] = y

```cpp
(1)朴素并查集：
    int p[N]; //存储每个点的祖宗节点
    // 返回x的祖宗节点 + 路劲压缩
    int find(int x) {
        if (p[x] != x) p[x] = find(p[x]);
        return p[x];
    }
     // 初始化，假定节点编号是1~n
    for (int i = 1; i <= n; i ++ ) p[i] = i;
    // 合并a和b所在的两个集合：就是把a的祖宗节点插到b的祖宗节点下面
    p[find(a)] = find(b);

(2)维护size的并查集：
    int p[N], size[N];
    //p[]存储每个点的祖宗节点, size[]只有祖宗节点的有意义，表示祖宗节点所在集合中的点的数量
    //注意：size[]只有祖宗节点才有意义，所以只用在合并操作中更新size

    // 返回x的祖宗节点
    int find(int x) {
        if (p[x] != x) p[x] = find(p[x]);
        return p[x];
    }
    // 初始化，假定节点编号是1~n
    for (int i = 1; i <= n; i ++ ) {
        p[i] = i;
        size[i] = 1;
    }
    // 合并a和b所在的两个集合：
    size[find(b)] += size[find(a)];
    p[find(a)] = find(b);

(3)维护到祖宗节点距离的并查集：距离可以映射为集合同类元素的某种性质
    int p[N], d[N];
    // p[]存储每个点的祖宗节点, d[x]存储x到p[x]的距离
    // 返回x的祖宗节点
    int find(int x) {
        if (p[x] != x) {
            //先进行路劲压缩，u是p[x]节点的祖宗节点，此时的p[x]还是x的父节点
            int u = find(p[x]);
            //上一步保证了,x的父节点是p[x],p[x]的父节点一定是祖宗节点
            //所以可以直接更新d[x]：加上p[x]节点到祖宗节点的距离
            d[x] += d[p[x]];
            //最后再更新x的父节点指向祖宗节点
            p[x] = u;
        }
        return p[x];
    }
    // 初始化，假定节点编号是1~n
    for (int i = 1; i <= n; i ++ ) {
        p[i] = i;
        d[i] = 0;
    }
    // 合并a和b所在的两个集合：
    p[find(a)] = find(b);
    d[find(a)] = distance; // 根据具体问题，初始化find(a)的偏移量
```

## 6 堆

- stl里的优先队列就是堆
- 堆是一个完全二叉树
- 小根堆：根结点小于等于孩子结点
- 大根堆：根结点大于等于孩子结点
- 存储：一维数组nums[i]
  - x的右孩子：nums[2x+1]
  - 根结点：nums[1], 从1开始避免特判，便于统一下面2x, 2x+1
  - x的左孩子：nums[2x]
- 两个基本操作：
  - up():lgn
  - down():lgn

以小根堆为例：

1. 插入一个数:`heap[++size] = x; up(size);`
2. 求集合当中的最小值:`heap[1];`
3. 删除最小值：使用尾节点覆盖第一个节点，`heap[1] = heap[size--]; down(1);`
4. 删除任意一个元素:`heap[k] = heap[size--]; down(k); up(k);`
5. 修改任意一个元素:`heap[k] = x; down(k); up(k);`

```cpp
// p[N]是元素值，h[N]存储堆中的值, h[1]是堆顶，x的左儿子是2x, 右儿子是2x + 1
// ph[k]存储第k个插入的点在堆中的位置，p--->h
// hp[k]存储堆中下标是k的点是第几个插入的，h---->p
// size表示当前堆中有多少元素
int h[N], ph[N], hp[N], size;

// 交换两个点，及其映射关系
void heap_swap(int a, int b) {
    swap(ph[hp[a]],ph[hp[b]]);
    swap(hp[a], hp[b]);
    swap(h[a], h[b]);
}
// 将节点u下沉到堆中的正确位置
void down(int u) {
    int t = u;
    // 把当前节点t分别和左右儿子比较，保证根节点为三者中最小
    if (u * 2 <= size && h[u * 2] < h[t]) t = u * 2;
    if (u * 2 + 1 <= size && h[u * 2 + 1] < h[t]) t = u * 2 + 1;
    if (u != t) {
        heap_swap(u, t);
        down(t);
    }
}
// 将节点u上升到堆中的正确位置
void up(int u) {
    while (u / 2 && h[u] < h[u / 2]) {
        heap_swap(u, u / 2);
        u >>= 1;
    }
}
// O(n)建堆
// 因为最后一层有n/2个节点，不用down.只需要把前n/2个节点down下来
for (int i = n / 2; i; i -- ) down(i);
```

## 7 Hash表

1. 一般哈希

    ```cpp
    (1) 拉链法
        int h[N], e[N], ne[N], idx;
        // 向哈希表中插入一个数
        void insert(int x) {
            int k = (x % N + N) % N;
            e[idx] = x, ne[idx] = h[k], h[k] = idx ++ ;
        }
        // 在哈希表中查询某个数是否存在
        bool find(int x) {
            int k = (x % N + N) % N;
            for (int i = h[k]; i != -1; i = ne[i])
                if (e[i] == x)
                    return true;
            return false;
        }
    (2) 开放寻址法
        // N一般开为输入个数的两到三倍，而且最好是一个质数
        // 初始化h[N]里面的值为一个不在输入范围内的数，比如0x3f3f3f3f
        int h[N];
        // 如果x在哈希表中，返回x的下标；如果x不在哈希表中，返回x应该插入的位置
        int find(int x) {
            int t = (x % N + N) % N;
            while (h[t] != null && h[t] != x) {
                t ++ ;
                if (t == N) t = 0;
            }
            return t;
        }
    ```

2. 字符串哈希

- 前缀哈希
- 把字符串看作就是一个数
- 默认是没有冲突的
- 处理字符串的利器，必须得会，快速判断两个字符串是不是相等(O(1))

```cpp
核心思想：将字符串看成P进制数，P的经验值是131或13331，取这两个值的冲突概率低
小技巧：取模的数用2^64，这样直接用unsigned long long存储，溢出的结果就是取模的结果
using ULL = unsigned long long;
ULL h[N], p[N]; // h[k]存储字符串前k个字母的哈希值, p[k]存储 P^k mod 2^64（就是进制的次方）
// 初始化
p[0] = 1;
for (int i = 1; i <= n; i ++ ) {
    // str[i]表示字符串中第i个字符，下标从1开始
    h[i] = h[i - 1] * P + str[i];
    p[i] = p[i - 1] * P;
}
// 计算子串 str[l ~ r] 的哈希值,是一个数
ULL get(int l, int r) {
    // 0---l---r，整个字符串看成一个p进制的数，左边是高位，右边是低位
    // 所以h[l---r] = h[0---r] - h[0---l-1]*p[r-l+1]
    return h[r] - h[l - 1] * p[r - l + 1];
}
```
